# Amalthea.jl Documentation
This is the documentation for [Amalthea.jl](https://github.com/vdiego28/Amalthea.jl), a performance-focused fork of [Luna.jl](https://github.com/LupoLab/Luna.jl) that replaces performance-critical numerical kernels with a native Rust backend. The [README](https://github.com/vdiego28/Amalthea.jl/blob/main/README.md) contains installation and quickstart instructions. This documentation is intended to lay out the physical model used and give deeper insight into the workings of Amalthea.jl. It is not yet complete--any contributions are welcome!

If you are new to Amalthea.jl, the most interesting page is probably [the simple interface](#The-simple-interface). Details about the actual equation Amalthea.jl solves can be found in [the numerical model](#The-numerical-model) section.
